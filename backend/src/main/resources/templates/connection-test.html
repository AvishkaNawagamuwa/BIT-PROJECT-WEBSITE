🐛 CORS Troubleshooting Checklist

  • ✅ Is frontend running on http://localhost:8000?
  • ✅ Check browser console (F12) for CORS errors

  • ✅ Check SecurityConfig.java allows CORS

  • ✅ Try in incognito mode (disable browser extensions)
  • ✅ Check MySQL database is running and accessible
